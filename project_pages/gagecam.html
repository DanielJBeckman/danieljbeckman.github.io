Gage-Cam

Gage-Cam is an AI-powered river stage estimation system developed using a U-Net architecture trained on synthetic imagery. The model detects exposed staff plate heights from images to estimate water level. The system was built from Raspberry Pi components, solar powered, and capable of edge/cloud processing. Gage-Cam replaced an earlier prototype, Cambot, which used an adapted Eigenfaces method but suffered from drift.

This project included full hardware prototyping, field testing, optics analysis, and model refinement. All image training data was self-generated to simulate varied water levels, lighting, and distortion effects.

Tech Used: Python · Keras · U-Net · OpenCV · Raspberry Pi · Synthetic Data · Edge Computing
