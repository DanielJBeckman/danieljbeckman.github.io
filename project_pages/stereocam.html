Stereo-Cam

A custom stereo vision rig for measuring river ice thickness using synchronized Raspberry Pi nodes and machine vision.

Overview

Stereo-Cam was a downward-looking stereo vision system designed to estimate ice thickness by capturing 3D geometry of river ice from a bridge-mounted rig. The system consisted of two waterproof Raspberry Pi A+ camera modules wired to a master controller node via Power-over-Ethernet (PoE), allowing precise image capture synchronization.

The master node handled asynchronous command-and-control and coordinated image acquisition from the Pi nodes. The captured stereo image pairs were analyzed using stereo vision algorithms to determine the distance from the cameras to the ice surface.

This value was then paired with readings from a bubbler sensor below the ice to calculate accurate ice thickness above datum.

Design Features

  • Dual synchronized Raspberry Pi A+ nodes
  • Master node with POE command system
  • Stereo image capture and depth estimation
  • Quick-attach mounting system to limit bridge exposure time
  • Field-tested under harsh winter conditions

Deployment

The system was successfully deployed for testing on a bridge during winter flood season. Despite operating correctly, the hardware was destroyed by a snowplow the day after its first test.
